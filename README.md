# 參孫打獅子 🦁

一個給主日學/兒童的聖經互動小遊戲,聚焦 **士師記 14:5-6** ——
參孫在亭拿的葡萄園,徒手撕裂向他吼叫的少壯獅子。

> 「……到了亭拿的葡萄園,見有一隻少壯獅子向他吼叫。**耶和華的靈大大感動參孫**,
> 他雖然手無器械,卻將獅子撕裂,如同撕裂山羊羔一樣……」(士師記 14:5-6)

這不是一個「比力氣」的遊戲——聖經沒說參孫多強壯,而說「耶和華的靈大大感動他」。
得勝的源頭是**神的靈**。遊戲在標題、勝利畫面與「🍯 蜂蜜與謎語」彩蛋裡,帶出這個信息。

> **🎮 線上直接玩**:https://hfpc-samson-game.netlify.app
> (可「加入主畫面」裝成 App、離線玩。也已收進 [聖經遊戲大廳](https://hfpc-bible-games.netlify.app) 的「⚔️ 戰爭闖關合輯」。)

## 怎麼玩

在場上自由走位,看獅子的節奏出手:

1. 獅子做出 **⚠ 預備動作**(後仰蓄力)——這時**紅線會一直追著你瞄準**,所以要持續走動,別站著不動。
2. 撲出前一刻,**紅線會定住**(變成更亮的實線)——這就是訊號!順著它**走位側身**離開那條直線,衝刺就撲空了(走位本身就是閃避)。
3. 獅子衝完**喘氣露出破綻 💢**——這時**靠近牠出手反擊**!
4. 抓準破綻反擊幾次,把獅子打倒,看「神的靈」的收尾。

### 操作

| 動作 | 鍵盤 | 手機/觸控 |
| --- | --- | --- |
| 🏃 走位(=閃避) | WASD / 方向鍵(持續按住) | **按住拖曳**畫面(虛擬搖桿) |
| 👊 反擊 | 空白 / Enter / J / K / F | **輕點**畫面(短按即放) |
| ⏸ 暫停 | P / Esc | 右上角按鈕 |
| 🔊 靜音 | M | 右上角按鈕 |

只有獅子**衝刺**撞到你才會扣心;單純走近不會。被撞到扣一顆心(共 5 顆),但有無敵閃爍、不會一下就結束;失敗了就再試一次,不會懲罰。
手機請**橫向**遊玩(直向會提示轉向,並自動全螢幕)。

## 怎麼啟動

- **最簡單**:雙擊 `start-game.bat`(第一次會自動安裝,約 1–2 分鐘;之後開瀏覽器即可玩)。
- **開發者**:

  ```
  npm install
  npm run dev        # 開瀏覽器到顯示的網址(預設 http://localhost:5273/)
  ```

## 改內容(給老師 / 牧者)

- **改經文、勝利/失敗的話、蜂蜜謎語彩蛋** → 只改 `src/scripture.js`(`LEVEL1`)。
- **調難度**(獅子多快、破綻多長、幾下打倒、幾顆心) → 只改 `src/config.js`(看 `PHASES`、`LION.maxHp`、`SAMSON.maxHearts`)。

## 離線 / 安裝到平板

`npm run build` 會產生 `site/`,可上傳到任何靜態主機(Netlify…)。裝成 PWA 後可離線玩,
適合教室沒網路的場合(建議裝好後先線上完整玩一輪,讓資源都快取下來)。
