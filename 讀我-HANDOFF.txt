這是什麼
--------
單一動作關的聖經小遊戲:參孫打獅子(士師記 14:5-6)。
純 vanilla ES modules + Canvas,零執行期相依、Web Audio 合成音效、可離線(PWA)。
架構沿用「約拿闖關」引擎。俯視角競技場 boss 戰:走位閃避 + 趁破綻反擊,6 下撕裂收尾。

★ 神學重點:聖經不說參孫多強壯,而說「耶和華的靈大大感動他」——力量出於神。
  不要把它改成「比力氣 / 練等變強」;不要加倚靠值能量條(使用者已明確決定不做)。

怎麼跑 / 測 / 建置(在這個資料夾)
---------------------------------
  npm install            第一次(裝 vite + sharp)
  npm run dev            開發,預設 http://localhost:5273/
  npm test               煙霧測試:文案 + 語法 + 狀態機/嵌入契約
  npm run test:offline   再加 build → PWA 離線就緒檢查
  npm run build          產生 site/(自寫 bundle,避開 Node 24 的 vite build 地雷)

本機地雷(務必遵守,詳見 CLAUDE.md):
  - 不要用 vite build(這台 Node 24 遞迴 cpSync 會無聲被殺);build 走 scripts/bundle-static.mjs。
  - .bat 一律純 ASCII + CRLF。
  - localhost 會自動 unregister Service Worker(改了才看得到);正式環境才註冊。
  - 新增 src/ 模組要加進 public/sw.js 的 CORE 清單(test:offline 會檢查)。

跨專案 Skills:一鍵裝齊全部(★ 換機必看)
------------------------------------------
Skill 真正生效在「使用者層」~/.claude/skills/,是機器本地、不會跟著任何 repo 走。
所有自製 skill(目前 30 個:遊戲引擎、闖關設計、卡片關、各式機制、神學、驗證/部署、
本遊戲新做的 蜂窩石頭/神蹟狂暴/地獄壞結局/向量美術 等)已打包成一個私有 repo,
換 PC / agape250 那台可「一鍵裝齊」:

  Skill 合輯 repo(私有):
    https://github.com/summer09201017-cloud/hfpc-claude-skills

  方式 B(推薦,Claude 原生、可更新):在 Claude Code 裡輸入
    /plugin marketplace add summer09201017-cloud/hfpc-claude-skills
    /plugin install hfpc-bible-game-skills@hfpc-skills
    之後更新:/plugin marketplace update hfpc-skills

  方式 A(零門檻,雙擊即裝):
    git clone https://github.com/summer09201017-cloud/hfpc-claude-skills.git
    然後雙擊 install.bat(Windows)或 bash install.sh(Mac/Linux/Git Bash)
    → 自動複製進 ~/.claude/skills/,重開 Claude 即可。

  ※ 私有 repo:新機要先 gh auth login(或用 GitHub 帳號登入)才能 clone/裝市集。
  ※ 新增/改 skill 的流程:在來源機 ~/.claude/skills 編好 → 同步進該 repo 的
     plugins/hfpc-skills/skills/ → git push;其他機 /plugin marketplace update
     或 git pull 後重跑 install。

本遊戲特別相關的 skill:
  ship-game-online      單機遊戲一條龍上架(git→GitHub→Netlify→驗證→點亮卡)
  add-to-collection     把一關加進大廳合輯卡片
  arcade-game-kit       零相依 Canvas 闖關引擎(本遊戲的底)
  arena-pickups-hazards / boss-escalation / corruption-bad-ending /
  canvas-vector-characters  ← 本遊戲這幾輪做出來的機制與美術,已抽成可重用 skill
